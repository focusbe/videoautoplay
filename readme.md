|   |  IOS微信   | 安卓微信  |  IOS携程   | 安卓携程  | PC 端 |
|  ----  |  ----  | ----  |  ----  | ----  | ---- |

# 使用方法
## 方法1:直接调用
1. 下载main.js
2. 调用
```javascript
vieoAutoPlay("#videoFrame");
```

## 方法2:npm
1. 
```
npm install --save-dev videoauto
yarn add videoauto
```

2. 调用
```javascript
const videoAutoPlay = require("videoauto");
vieoAutoPlay("#videoFrame");
```
